## 🛠️ How to Use This Cheat Sheet

### 1. Compilation
You can compile these cheat sheets using a local LaTeX environment or an online editor like Overleaf.

*   **Local Compilation**: Navigate to the specific cheat sheet folder, ensure you have a TeX distribution installed (e.g., TeX Live, MacTeX), and run:
    ```bash
    cd EE6497_Quiz1_Cheatsheet # or other folders
    pdflatex main.tex
    ```
    *> Note: Run the command multiple times if necessary to resolve cross-references.*

*   **Overleaf**: Simply upload the specific project folder to Overleaf and set the compiler to **pdfLaTeX** in the project menu.

### 2. Customization
The projects are split into content (`main.tex`) and configuration (`preamble.tex`) to facilitate updates:

*   **Adding Content (`main.tex`)**:
    The document uses a 4-column layout (adjustable). To add a new topic, use the defined commands in the preamble.
    ```latex
    \bulletPoint{Topic Name}:
    Your content here...
    ```

*   **Modifying Styles (`preamble.tex`)**:
    Edit this file to change package imports, page margins, fonts, or custom command definitions.
