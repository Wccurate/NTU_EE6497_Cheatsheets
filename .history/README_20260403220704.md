## 🛠️ How to Use This Cheat Sheet

### 1. Compilation
You can compile this cheat sheet using a local LaTeX environment or an online editor like Overleaf.

*   **Local Compilation**: Ensure you have a TeX distribution installed (e.g., TeX Live, MacTeX). Run the following command in your terminal:
    ```bash
    pdflatex main.tex
    ```
    *> Note: Run the command multiple times if necessary to resolve cross-references.*

*   **Overleaf**: Simply upload the project folder to Overleaf and set the compiler to **pdfLaTeX** in the project menu.

### 2. Customization
The project is split into content (`main.tex`) and configuration (`preamble.tex`) to facilitate updates:

*   **Adding Content (`main.tex`)**:
    The document uses a 4-column layout. To add a new topic, use the `\bulletPoint` command defined in the preamble:
    ```latex
    \bulletPoint{Topic Name}:
    Your content here...
    ```

*   **Modifying Styles (`preamble.tex`)**:
    Edit this file to change package imports, page margins, fonts, or custom command definitions.
